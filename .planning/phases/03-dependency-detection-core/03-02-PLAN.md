    **Function signature (per D-17):**
    ```python
    def classify_pair(
        question_a: str,
        question_b: str,
        event_id_a: str | None = None,
        event_id_b: str | None = None,
        weights: dict[str, float] | None = None,
        thresholds: dict[str, float] | None = None,
    ) -> DependencyResult:
    ```

    **Implementation (per D-13, D-15):**
    1. Default `weights` to `DEFAULT_WEIGHTS` if None, `thresholds` to `DEFAULT_THRESHOLDS` if None
    2. Preprocess both questions: `tokens_a = _preprocess(question_a)`, `tokens_b = _preprocess(question_b)`
    3. Compute all 5 signals:
       - `jaccard = _jaccard_similarity(tokens_a, tokens_b)`
       - `implication = _keyword_implication(question_a, question_b)` (uses original strings, not preprocessed tokens -- Pitfall 3)
       - `numeric = _numeric_relation(question_a, question_b)` (original strings)
       - `temporal = _time_relation(question_a, question_b)` (original strings)
       - `event_bonus_val = _event_bonus(event_id_a, event_id_b)`
    4. Compute weighted score (per D-13):
       ```python
       score = (
           weights["jaccard"] * jaccard
           + weights["implication"] * implication
           + weights["numeric"] * numeric
           + weights["temporal"] * temporal
           + weights["event_bonus"] * event_bonus_val
       )
       ```
    5. Classify (per D-15):
       - `score >= thresholds["subset"]` -> label = "subset"
       - `score >= thresholds["related"]` -> label = "related"
       - otherwise -> label = "independent"
    6. Return `DependencyResult(label=label, score=score, jaccard=jaccard, implication=implication, numeric=numeric, temporal=temporal, event_bonus=event_bonus_val)`

    **Weight tuning note (per D-16):** If the validation set tests fail because threshold boundaries are wrong, adjust DEFAULT_WEIGHTS or DEFAULT_THRESHOLDS to make the validation set pass. The research suggests:
    - Deadline-variant pairs (same event) should score >= 0.70 (subset): high jaccard (~0.5-0.8) + temporal (1.0) + event_bonus (1.0) = well above 0.70
    - Candidate-variant pairs (same event) should score >= 0.35 but < 0.70 (related): high jaccard (~0.8) + event_bonus (1.0) but NO implication/temporal/numeric = jaccard*0.30 + event*0.15 = ~0.39
    - Independent pairs (different events) should score < 0.35: low jaccard, no signals, no event bonus
